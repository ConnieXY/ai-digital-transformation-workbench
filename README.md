# 企业 AI 数智化转型工作台

> 从企业效能诊断、行业方案生成，到具体业务场景落地，帮助企业把模糊的 AI 转型诉求，转化为可执行的方案。

一个面向 B2B / 咨询场景的 AI 数智化转型工作台 Demo —— 以「**Diagnose → Design → Deliver**」为方法主线，串起诊断、方案、落地三个模块，并以**制造业质量异常闭环**作为完整的场景化样板。

🔗 **在线体验**：_部署后补充链接_

---

## 为什么做这个

企业对 AI 转型常有诉求，却卡在「不知从何下手」：现状不清、方案空泛、难以落地。本工作台把抽象目标拆解为结构化、可执行的步骤，让每一步都有清晰产出。

落地场景选「制造业质量异常闭环」，源自真实问题——服务过汽车电子、半导体等制造业客户，「质量异常处理慢、知识难沉淀、跨部门协同低效、管理层数据不可视」是高频共性痛点。

## 三大模块与方法链路

| 步骤 | 模块 | 能做什么 |
| --- | --- | --- |
| **Diagnose** 诊断企业现状 | 企业效能诊断 | 6D 成熟度问卷 → 综合评分与等级 → 核心瓶颈 → 推荐场景机会地图 → 30/60/90 落地路线图 |
| **Design** 生成行业方案 | 行业解决方案生成 | 行业/客户/痛点输入 → 痛点地图、解决方案组合、角色化价值主张、一页纸方案 |
| **Deliver** 落地业务场景 | 制造业质量异常闭环 | 异常上报 → AI 结构化与根因分析 → 闭环任务看板 → 复盘报告 → 知识沉淀 |

三个模块的数据可贯通：诊断结果可带入行业方案，方案可衔接到制造业落地 Demo。

## 功能亮点

- **完整闭环**：三条业务线均为端到端可走通的闭环，而非孤立页面。
- **一键示例**：每个模块都支持「填入示例 / 查看示例结果」，无需填表即可看到完整产出。
- **结构化产出**：诊断报告、机会地图、解决方案表、复盘报告等均为可对客的专业格式。
- **统一设计语言**：一致的卡片、按钮、表格、标签与导航，面向 B2B SaaS 的清爽专业风格。

## 技术栈

- [Next.js 14](https://nextjs.org/)（App Router）
- React 18 + TypeScript
- Tailwind CSS
- 由 [Claude Code](https://claude.com/claude-code) 协助，以 Vibe Coding 方式从 0 到 1 实现

> 当前为演示版本：分析与生成结果基于内置规则（mock），暂未接入真实 AI 模型；数据保存在浏览器本地 `localStorage`，不上传服务器。

## 本地运行

```bash
npm install
npm run dev
# 打开 http://localhost:3000
```

构建生产版本：

```bash
npm run build
npm run start
```

## 目录结构

```
app/            页面与路由（诊断 / 方案 / 制造业 Demo / 项目说明）
components/     UI 组件（按模块拆分）
data/           各模块的结构化数据与类型定义
lib/            评分、方案生成、异常分析、任务生成等规则逻辑
```
